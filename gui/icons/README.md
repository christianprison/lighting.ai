# Icons für Instrumente

Platzieren Sie hier die generierten Icon-Bilder für die Instrumente.

## Dateinamen-Konvention:

Die Icons sollten nach Instrument benannt werden:
- `bassdrum.png`
- `snare.png`
- `tom1.png`, `tom2.png`, `tom3.png`
- `overhead1.png`, `overhead2.png`
- `bass.png`
- `lead_guitar.png`
- `rhythm_guitar.png`
- `vocals_frontman.png`
- `vocals_bassist.png`
- `vocals_guitarist.png`
- `synthesizer.png`

## Format:

- PNG mit transparentem Hintergrund
- Empfohlene Größe: 200x200 Pixel
- Nur Umriss/Zeichnung, keine Füllung (wird dynamisch basierend auf Meter-Level hinzugefügt)

Die Icons werden automatisch geladen, wenn sie im `gui/icons/` Verzeichnis vorhanden sind.
Die Pfade werden in der Datenbank-Konfiguration gespeichert.
